# 🎯 AI Course Recommendation System

An intelligent web application that recommends courses based on user preferences using Machine Learning.

## 🚀 Features
- User authentication (login/register)
- Personalized course recommendations
- Admin dashboard
- ML-based recommendation engine
- Clean UI with Flask templates

## 🛠 Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS (Jinja2 Templates)
- Database: SQLite
- Machine Learning: Pandas, Scikit-learn

## 📂 Project Structure
```
├── app.py
├── ml_model.py
├── students.db
├── templates/
├── static/
├── courses.csv
├── jobs.csv
```

## ⚙️ Installation

```bash
git clone https://github.com/YOUR_USERNAME/ai-course-recommendation-system.git
cd ai-course-recommendation-system
pip install -r requirements.txt
python app.py
```

## 🌐 Deployment
This project can be deployed on platforms like Render or Railway.

## ⚠️ Note
- SQLite is used for development. For production, use PostgreSQL or MongoDB.
- Ensure proper environment setup before deployment.
